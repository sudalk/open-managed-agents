Quickstart

Get started with Managed Agents in minutes.

1. Set your API key

Enter your API key in the sidebar below. This is stored in your browser only.

2. Create an agent

Go to Agents and click New agent. Choose a model and system prompt.

3. Create an environment

Go to Environments and create a container template.

4. Start a session

Go to Sessions, create a new session binding your agent + environment, and start chatting.

Or use the API directly

# Create agent
curl -X POST /v1/agents -H "x-api-key: $KEY" \
  -d '{"name":"Assistant","model":"claude-sonnet-4-6"}'

# Create environment
curl -X POST /v1/environments -H "x-api-key: $KEY" \
  -d '{"name":"default","config":{"type":"cloud"}}'

# Create session
curl -X POST /v1/sessions -H "x-api-key: $KEY" \
  -d '{"agent":"agent_...","environment_id":"env_..."}'

# Send message
curl -X POST /v1/sessions/$ID/events -H "x-api-key: $KEY" \
  -d '{"events":[{"type":"user.message","content":[{"type":"text","text":"Hello"}]}]}'

# Stream response
curl -N /v1/sessions/$ID/events -H "x-api-key: $KEY"
